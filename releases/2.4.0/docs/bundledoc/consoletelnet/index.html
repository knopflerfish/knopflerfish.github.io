Bundle: consoletelnet
Version 2.0.4

Console Telnet

This bundle uses the KF console to export commands on a telnet server.

Description

The default port is 23, default login "admin", password "admin"

The consoletelnet bundle can be configured using system properties:

org.knopflerfish.consoletelnet.user
User login name for remote access.Default is admin
org.knopflerfish.consoletelnet.pwd
User login name for remote access. Default is admin
org.knopflerfish.consoletelnet.port Port number telnet server will listen on Default is 23
