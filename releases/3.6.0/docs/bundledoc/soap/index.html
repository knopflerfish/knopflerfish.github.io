SOAP and WebServices in Knopflerfish

There are three different SOAP implementations for Knopflerfish:

  • Axis (also known as Axis 1),
  • kSOAP,
  • Axis2.

Axis 1

The Knopflerfish Axis 1 port provides SOAP/Web service access to any OSGi bundle, both for exporting OSGi services as Web Services and for importing web services into an OSGI framework.

The Knopflerfish Axis 1 port is the oldest one. The bundle is around 1.3 M bytes in size. The following bundles are part of it:

  • axis-osgi - The SOAP server implementation.
  • soapobject - Example bundle that publishes a WebService, remoteFW.
  • soapclient - Example of a soap client that uses the remoteFW.
  • soap_desktop - A SOAP service browser for the KF Swing Desktop.
  • soap_remotefw - A complete remote OSGi framework, using SOAP. Acting either in client mode or in server mode.
  • demos - SOAP demo bundles are available on https://www.knopflerfish.org/svn/knopflerfish.org/trunk/osgi/bundles_opt/soap/demos/.

kSOAP

The kSOAP implementation is smaller than Axis 1 (only 138 k bytes) The following subdirectories are part of it:

  • ksoap-osgi - The kSOAP server implementation.
  • ksoap_remotefw - A complete remote OSGi framework, using SOAP. Acting either in client mode or in server mode.

Axis 2

The Knopflerfish Axis2 port is the most recent one. Event though it is a stripped down version of the Axis2 Web Application it is fairly large, around 7 M bytes. The following bundles are part of it:

  • axis2-osgi - The Axis2 SOAP server implementation.
  • xerces - Apache Xerces-J XML parsing support needed by the axis2-osi bundle.
  • xalan - Apache Xalan-J XML transformation support needed by the axis2-osgi bundle.
  • axis2_soapobject - A bundle that publishes a Web-Service. Part of the KF-regression test-suite. Source code is available at https://www.knopflerfish.org/svn/knopflerfish.org/trunk/osgi/bundles_test/regression_tests/axis2_soapobject/
  • axis2_soapclient_adb - A SOAP client (using unwrapped ADB style stub) bundle that accesses the service published by axis2_soapobject. Part of the KF-regression test-suite. Source code is available at https://www.knopflerfish.org/svn/knopflerfish.org/trunk/osgi/bundles_test/regression_tests/axis2_soapclient_adb/
